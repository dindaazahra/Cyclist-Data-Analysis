# Cyclist Dashboard
---
### Setup Environment
```
conda create --name cyclist-dashboard python=3.9
conda activate cyclist-dashboard
pip install streamlit pandas matplotlib seaborn
```
---
### Run Streamlit App
```
streamlit run cyclist_dashboard.py
```
